#  House Price Prediction using Machine Learning

##  Overview

This project predicts house prices using Linear Regression based on features like living area, bedrooms, and bathrooms.

##  Features

* Data preprocessing
* Model training
* Model evaluation (MSE, R² Score)
* Price prediction for new data

##  Tech Stack

* Python
* Pandas
* Scikit-learn

##  Dataset

Kaggle House Prices Dataset

##  How to Run

1. Clone the repository
2. Install requirements
3. Run the notebook
